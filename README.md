
Tên thư mục/file	Vai trò	Tại sao nên backup?
hypr/	Hyprland WM	Chứa phím tắt, cấu hình hiệu ứng, giao diện Window Manager của bạn.
waybar/	Status bar	Thanh trạng thái hiển thị pin, mạng, CPU, nút thông báo và nút nguồn.
swaync/	SwayNotificationCenter	Bảng điều khiển trung tâm và giao diện thông báo.
rofi/	App Launcher / Menus	Menu ứng dụng, menu nút nguồn và lịch sử clipboard.
kitty/	Terminal	Cấu hình font chữ, màu sắc, độ trong suốt của cửa sổ terminal.
wallpaper_custom/	Hình nền	Chứa ảnh nền bạn đang gọi từ file cấu hình Hyprland.
fastfetch/	System Info	Cấu hình hiển thị thông tin hệ thống khi mở Terminal.
MangoHud/	Performance Overlay	Cấu hình thông số FPS/Nhiệt độ khi chơi game.
easyeffects/	Audio Equalizer	Bộ lọc âm thanh hiệu ứng loa/tai nghe (chỉnh tay cấu hình này rất mất thời gian).
fcitx5/	Input Method	Cấu hình bộ gõ tiếng Việt Bamboo.
nwg-displays/	Monitor Layout	Vị trí màn hình phụ, độ phân giải và tần số quét (Hz).
chrome-flags.conf	Google Chrome flags	Chứa cờ tính năng (như cuộn chuột giữa) của Chrome.
mimeapps.list	Default Apps	Danh sách cài đặt ứng dụng mặc định để mở file.
mpd/	Music Player	Cấu hình trình phát nhạc nền (nếu bạn sử dụng).
